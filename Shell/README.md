1. Записать содержимое архива на CD-R (не перезаписываемый) в папку [Буква диска]:\Shell 
2. Запустить файл start.bat от имени администратора
3. После перехзагрузки в безопасном режиме выполнить вход учетной записью администратора
4. По окончанию проверки утилита перезагрузит машину в стандартный режим.

P.S.: Использовать на свой страх и риск, автор скрипта не несёт ответственности за уровень подготовки пользователей скрипта.
